Tableau



Tableau is increasingly useful for data scientists as it includes more fine-grained functions. With Tableau, we can do multiple and amazing things. For example, we can generate graphs, maps, charts, visualizations, and most importantly, dashboards and stories. This tool is an ally for data scientists, data analysts, and anybody whose concern is to present results and data professionally and smoothly. From my perspective, I think Tableau it’s powerful when your data was wrangled and cleaned using R or Python, for example, to have a data frame ready for creating visualizations. In that scenario, Tableau is one of the best tools to produce professional graphics, even when we don’t know a lot of programming languages. Tableau is fast and user-friendly to use understand and use it instantly without prior knowledge.
